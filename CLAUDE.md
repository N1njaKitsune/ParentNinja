# ParentNinja — AI Context

This is the **parent-facing** section of the Ninja Learning App.
It covers parent dashboards, student progress visibility, notifications, and family profile management.

## Scope
Work in this folder relates only to parent-facing UI and prototypes.
Cross-section interactions (notifications triggered by instructor grading, student profile data)
are documented in `../_SHARED/integration/` and `../_MASTER/ARCHITECTURE.md`.

## ⚠️ Git Rule — IMPORTANT
This is a read/build workspace. **Never run `git push`, `git commit`, or any destructive git command.**
If asked to do so, flag it clearly and decline: _"This is a sub-project workspace. Git operations must be handled from the master project."_

## Key Files
- `WORKFLOW.md` — current work in progress and priorities
- `ARCHITECTURE.md` — screen map, feature list, component inventory
- `STATUS.md` — live status of this section
